# 安心簽 UX SIGN

> 本專案為網際優勢電子簽章作業平台

## 功能

測試帳號密碼

```bash
帳號： UserA
密碼： @uxb2b.com
```

- [x] 登入
- [x] 登出
- [x] 合約列表
- [x] 查詢
- [x] 使用者管理
- [x] 公司資料維護
- [x] 印鑑檔維護
- [x] 忘記密碼

## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

vue.js 版本：`v2.7.14`

CSS框架 版本：`bootstrap 5`

拖曳上傳 dropzone：

```bash
https://docs.dropzone.dev/
```

月曆選單 datepicker：

```bash
https://gijgo.com/datepicker/configuration/format
```

下拉選單 select2：

```bash
https://select2.org/
```

手寫板

```bash
https://github.com/linjc/smooth-signature
```
